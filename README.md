# Roblox Placement System

A simple and flexible object placement system for Roblox, designed for building mechanics and interactive environments.

## Features

- Grid snapping support
- Object rotation controls
- Real-time placement preview
- Collision checking before placement
- Client and server handling
- Modular and easy to extend

## How it works

- Player selects an object to place
- A preview (ghost) follows the cursor
- Object snaps to grid (if enabled)
- Player rotates and positions the object
- Placement is validated (collision, rules)
- Object is placed in the world
